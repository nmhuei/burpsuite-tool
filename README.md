# Burpsuite Tool (Burp Bridge)

Standalone Burp automation bridge extracted from `security_audit_tool`.

## Features
- Burp request/response ingest bridge (`collector.py`)
- Heuristic analyzer (`analyzer.py`)
- Markdown report generator (`reporter.py`)
- Burp Jython extension (`burp_extender.py`)
- One-command timetable pipeline (`auto_timetable.sh`)

## Quick start
```bash
./run_mvp.sh
```
